This repository is intended for testing image-to-image translation architectures on ALSF pilot data.

## Overview

The goal of this project is to explore and evaluate image-to-image translation architecutres performance across various seedning densities pilot data from the Alex's Lemonade Stand Foundation (ALSF) program.
